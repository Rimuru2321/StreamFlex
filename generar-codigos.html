⭐ Generador de Códigos Premium

Genera códigos únicos para darle a cada pagador de Patreon/Ko-fi

⚠️ Cómo funciona:
1. Genera los códigos aquí
2. Añádelos a Firestore (instrucciones abajo)
3. Dale un código por persona que pague
4. Cuando lo usen → se marca used: true en Firestore → nadie más puede usarlo
📋 Añadir códigos a Firestore: Ve a console.firebase.google.com → Tu proyecto → Firestore → Colección premiumCodes → Añadir documento.

Por cada código, crea un documento con ID = el código y estos campos: 
{
  "used": false,
  "createdAt": "2026-01-01",
  "note": "Patreon - enero 2026"
}
